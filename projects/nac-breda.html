🥅 Smart Player Recruitment for NAC Breda

November 2023 - January 2024

Client: NAC Breda Professional Football Club

Project Overview

Developed a data-driven scouting tool for NAC Breda to enhance their player acquisition strategy using machine learning and advanced analytics. This project transformed how the club identifies talent, moving from subjective assessments to data-backed decisions while respecting the intuitive aspects of traditional scouting.

The Challenge

NAC Breda needed to compete with larger clubs despite limited scouting resources:

• Analyzing thousands of potential players across multiple leagues
• Identifying undervalued talent before competitors
• Predicting player development potential
• Ensuring GDPR compliance with player data
• Balancing analytics with traditional scouting wisdom
• Creating actionable insights for non-technical staff

My Approach

I developed a comprehensive analytics platform combining multiple data science techniques:

1. Data Integration: Unified data from Opta, WyScout, and internal scouting reports
2. Feature Engineering: Created 100+ performance metrics beyond basic statistics
3. Machine Learning: Built ensemble models for player classification and potential prediction
4. Interactive Dashboards: Developed Streamlit app for scouts to explore insights
5. Mathematical Optimization: Used linear programming for squad composition

Technical Implementation

The system combines supervised and unsupervised learning for comprehensive player analysis: 


# Player Recruitment Analytics System
import pandas as pd
import numpy as np
from sklearn.ensemble import GradientBoostingRegressor
from sklearn.decomposition import PCA
from sklearn.cluster import KMeans
import streamlit as st

class PlayerScoutingSystem:
    def __init__(self):
        self.performance_model = GradientBoostingRegressor(
            n_estimators=200,
            learning_rate=0.05,
            max_depth=5
        )
        self.potential_model = self.build_potential_predictor()
        self.style_clusterer = KMeans(n_clusters=7)
        
    def analyze_player(self, player_data):
        # Performance Rating
        current_level = self.performance_model.predict(
            player_data[self.performance_features]
        )
        
        # Potential Prediction (considering age curve)
        age_factor = self.calculate_age_curve(player_data['age'])
        potential = self.potential_model.predict(
            player_data[self.potential_features]
        ) * age_factor
        
        # Playing Style Classification
        style_vector = self.extract_style_features(player_data)
        style_cluster = self.style_clusterer.predict(style_vector)
        
        # Value Assessment
        market_value = self.estimate_value(
            current_level, potential, player_data['contract_remaining']
        )
        
        return {
            'current_rating': current_level,
            'potential_rating': potential,
            'playing_style': self.style_labels[style_cluster],
            'estimated_value': market_value,
            'similar_players': self.find_similar(style_vector),
            'strengths': self.identify_strengths(player_data),
            'development_areas': self.identify_weaknesses(player_data)
        }
    
    def squad_optimization(self, available_players, budget, requirements):
        # Linear programming for optimal squad composition
        from scipy.optimize import linprog
        
        # Maximize total quality within budget
        c = -available_players['quality_score'].values
        A_ub = available_players['price'].values.reshape(1, -1)
        b_ub = [budget]
        
        # Position constraints
        for position in requirements:
            constraint = (available_players['position'] == position)
            A_ub = np.vstack([A_ub, -constraint.astype(int)])
            b_ub.append(-requirements[position])
        
        result = linprog(c, A_ub=A_ub, b_ub=b_ub, 
                        bounds=(0, 1), method='highs')
        
        return available_players[result.x > 0.5]

Results & Impact

The scouting tool delivered measurable improvements to NAC Breda's recruitment:

• Scouting Efficiency: 75% reduction in initial player screening time
• Hidden Gems: Identified 3 undervalued players signed for €2M total
• Success Rate: 80% of recommended players met performance expectations
• Cost Savings: €500K saved through better negotiation positioning
• Youth Development: Improved U21 player progression tracking
• GDPR Compliance: Full compliance with automated data governance

The system is now integrated into NAC Breda's scouting workflow, with scouts using it daily to support their decision-making process.

Key Learnings

Domain Knowledge Matters: Understanding football tactics and scouting philosophy was as important as technical skills. Regular sessions with scouts improved model relevance significantly.

Trust Through Transparency: Scouts initially skeptical of "replacing intuition with algorithms" became advocates when they understood the tool augments rather than replaces their expertise.

Data Quality Issues: Football data is messier than expected - handling inconsistent player names, position classifications, and missing data required significant preprocessing.

Ethical Considerations: Careful consideration of fairness in player evaluation, especially regarding different playing styles and leagues.
